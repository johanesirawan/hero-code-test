# Hero-code-test

## Setup

- Require npm install in wwwroot
- Require to setup user secret for API Key and Base Url Endpoint with following format:
```sh
{
  "HeroApiSettings": {
    "BaseUrl": "https://example.com",
    "APIKey": "EXAMPLE_API_KEY"
  }
}

```
